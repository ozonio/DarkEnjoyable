## Installation ##

- Download the `DarkEnjoyable.icls` file
- In `File` -> `Settings` -> `Editor` -> `Colors Scheme` Select the gear and then select `Import Scheme`
- Select the file `DarkEnjoyable.icls` you downloaded
- Press `Apply` button
- Enjoy
